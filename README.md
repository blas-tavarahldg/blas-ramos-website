# blasramosv9Final - WordPress Theme

## Installation
1. Download `blasramosv9Final.zip`
2. In WordPress Admin: Appearance > Themes > Add New > Upload Theme
3. Select the ZIP file and click Install Now
4. Activate the theme
5. Set your homepage to a static page (Settings > Reading > A static page)

## Features
- Custom cursor system (dot + ring)
- Animated loader with logo
- 8-section single-page layout
- Story filtering system with categories
- Scroll reveal animations (IntersectionObserver)
- Mobile hamburger menu
- Contact form with validation
- Smooth scroll navigation

## Fonts
- Playfair Display (headings)
- DM Sans (body text)
